{{% blocks/products/pf/agp/feature-section-col title="How to Convert XBRL to SEC html report" %}}
1. Load input XBRL file using [XbrlDocument class](https://apireference.aspose.com/finance/net/aspose.finance.xbrl/xbrldocument).
2. Set [SaveOptions](https://apireference.aspose.com/finance/net/aspose.finance.xbrl/saveoptions) for the output file by choosing the [SaveFormat.SEC_HTML_REPORT](https://apireference.aspose.com/finance/net/aspose.finance.xbrl/saveformat).
3. Call the [Save method](https://apireference.aspose.com/finance/net/aspose.finance.xbrl.xbrldocument/save/methods/2) by providing the target file and relevant SaveOptions as parameters.
