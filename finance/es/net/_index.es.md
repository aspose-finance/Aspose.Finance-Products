<p align="justify"> Aspose.Finance for .NET es una biblioteca flexible para el procesamiento de formatos relacionados con las finanzas. API puede crear, leer y validar fácilmente los formatos XBRL y iXBRL del lenguaje extensible de informes comerciales, así como crear OFX archivos de solicitud y respuesta en diferentes formatos. Los desarrolladores pueden desarrollar software habilitado para XBRL con facilidad o pueden usarlo dentro de cualquier sistema de información financiera digital existente para crear informes con XBRL. Es un API extensible, fácil de usar e independiente de cualquier otro software.</p>
