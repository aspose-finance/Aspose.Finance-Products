<p align="justify"> Aspose.Finance for .NET هي مكتبة مرنة لمعالجة التنسيقات المتعلقة بالتمويل. يمكن API بسهولة إنشاء وقراءة والتحقق من صحة بيانات لغة الإبلاغ عن الأعمال القابلة للقراءة والتحقق من صحتها بسهولة XBRL و iXBRL بالإضافة إلى إنشاء OFX ملفات الطلب والاستجابة بتنسيقات مختلفة. يمكن للمطورين تطوير XBRL برنامج مكّن بسهولة أو يمكنهم استخدامه ضمن أي نظام تقارير مالية رقمية موجود لإنشاء تقارير باستخدام XBRL. إنه قابل للتوسعة وسهل الاستخدام ومستقل API مستقل عن أي برنامج آخر.</p>
