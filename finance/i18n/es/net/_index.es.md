<p align="justify"> Aspose.Finance for .NET es una biblioteca flexible para el procesamiento de formatos relacionados con las finanzas. API puede crear, leer y validar fácilmente formatos eXtensible Business Reporting Language XBRL y iXBRL, así como crear OFX archivos de solicitud y respuesta en diferentes formatos. Los desarrolladores pueden desarrollar XBRL software habilitado con facilidad o pueden usarlo dentro de cualquier sistema de informes financieros digitales existente para crear informes con XBRL. Es un API extensible, fácil de usar e independiente de cualquier otro software.</p>
