<p align="justify"> Aspose.Finance for .NET adalah perpustakaan fleksibel untuk pemrosesan format terkait keuangan. API dapat dengan mudah membuat, membaca, dan memvalidasi bahasa pelaporan bisnis yang dapat diperluas XBRL dan iXBRL format serta membuat permintaan OFX dan file respons dalam format yang berbeda. Pengembang dapat mengembangkan XBRL perangkat lunak yang diaktifkan dengan mudah atau dapat digunakan dalam sistem pelaporan keuangan digital yang ada untuk membuat laporan dengan XBRL. Ini adalah extensible, mudah digunakan dan mandiri API independen dari perangkat lunak lain.</p>
